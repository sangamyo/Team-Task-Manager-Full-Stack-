🎯 3-STEP DEPLOYMENT (5 MINUTES)
═══════════════════════════════════════════════════════════════════════════════

STEP 1: Deploy Backend to Railway (2 min)
  1. Go to railway.app/dashboard
  2. Create new project → Connect GitHub
  3. Select: sangamyo/Team-Task-Manager-Full-Stack-
  4. Set root directory: /api
  5. Add environment variables (see PRODUCTION_DEPLOYMENT.md)
  6. Click Deploy
  7. Copy Railway URL (you'll need it)

STEP 2: Deploy Frontend to Vercel (2 min)
  1. Go to vercel.com/dashboard
  2. Create new project → Import repository
  3. Select: sangamyo/Team-Task-Manager-Full-Stack-
  4. Set root directory: ./web
  5. Add NEXT_PUBLIC_API_URL = Railway URL (from Step 1)
  6. Click Deploy
  7. Copy Vercel URL

STEP 3: Update Backend with Frontend URL (1 min)
  1. Go to Railway Dashboard
  2. Update CLIENT_URL = Vercel URL
  3. Redeploy
  4. DONE! ✅
